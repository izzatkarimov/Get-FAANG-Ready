## <p align="center">Month 1</p>
### The Following is the Plan for Month 1

- [ ] Start a New Project
The Project should follow the  Software Development Cycle
The Project should not take less than 2 weeks to make, meaning that it should be relatively challenging
- [ ] Learn Data Structures & Algorithms

## <p align="center">Month 2</p>
#### The Following is the Plan for Month 2

- [ ] Finish the Project
- [ ] Start Blind 75 LeetCode Questions (One a Day)

## <p align="center">Month 3</p>
#### The Following is the Plan for Month 3

- [ ] Start the Second Project
- [ ] Start AWS Cloud Practitioner Certificate Preparation
- [ ] Start Oracle Certified Professional, Java SE 8 Programmer Preparation

- [ ] Get the Oracle Certified Professional, Java SE 8 Programmer Preparation

#### The Following is the Plan for Month 5

- [ ] Finish the Second Project
- [ ] Finish NeetCode 150

#### The Following is the Plan for Month 6

- [ ] Start Applying
